# AnytimeTS

**AnytimeTS: A Unified Anytime Learning Framework for Multi-Horizon Time Series Forecasting**

> 📌 **Status:** This repository accompanies a paper currently under review. The source code will be released upon acceptance of the paper.

## Overview

AnytimeTS is a unified anytime learning framework for multi-horizon time series forecasting. Further details, including the full methodology and experimental results, are available in the paper.

## Code Release

The source code and experiment scripts will be made publicly available in this repository once the associated paper has been accepted for publication. Please check back later, or watch/star this repository to be notified of updates.
